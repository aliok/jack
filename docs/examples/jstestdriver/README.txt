

To run the tests:

1. Type this on the command line: 
   java -jar JsTestDriver.jar --port 9876
2. Open this URL in your browser: 
   http://localhost:9876/capture
3. On the command line again:
   java -jar JsTestDriver.jar --tests all


An introduction to JsTestDriver can be found here:
http://code.google.com/p/js-test-driver/wiki/GettingStarted
